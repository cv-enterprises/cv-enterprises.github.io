Six Million Stories

Dylan Wilbur

This is a proof of concept application which enables users to create ‘counters’ that persist over the course of each visit to the site. Each page visit generates a different Holocaust victim story, giving a voice to those lost in time. Built as an educational tool for teachers, this application helps students visualize the enormity of the lives lost in the Holocaust, as well as individualizes the victims. With each page visit, a new Holocaust victim story is presented.
